## Setup and Installation

### 1. Environment Variables
This project uses a shared `.env` file for API keys. For security reasons, the actual `.env` file is not uploaded to GitHub. 
* See `.env.example` for the required variable names.
* Create your own `.env` file locally and add your keys.

### 2. Virtual Environment
Each day contains its own logic, but you can run them using a standard Python environment. 
* To keep the repository clean, all `venv/` folders are ignored by the global `.gitignore`.
